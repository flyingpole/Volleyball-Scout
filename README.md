# Volleyball Scout

Phone-first volleyball scouting app combining the serve receive opponent tracker with the serve receive and attacking tracker.

## Features

- Track one team or two teams.
- Turn serve receive tracking, attacking tracking, SR heat map, and haptics on or off with slide switches.
- Compact player list for phone use during play.
- Each player row has quick `SR` and `ATT` buttons; the scoring buttons open in a popup instead of taking permanent screen space.
- Serve receive scores: `0`, `1`, `2`, `3`.
- Attack scores: `+`, `.`, `-`, `T`, `t`; double-click `ATT` to record `.` as a standard attempt.
- Auto-disappearing undo confirmation after each score.
- Delete players entered by mistake.
- Libero sorting and highlighting.
- Local browser persistence.
- Print/PDF summary.

## Use Locally

Open `index.html` directly or serve this folder with any static web server.

For mobile installation behavior, host the folder on GitHub Pages, Netlify, Cloudflare Pages, or another static host.
